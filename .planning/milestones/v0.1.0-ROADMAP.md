**Status:** SHIPPED 2026-04-03

## Overview

Research-grade Python implementation of Dynamic Causal Modeling (DCM) for neuroimaging using Pyro for probabilistic inference. Three DCM variants (task-based, spectral, regression) implemented from first principles with full mathematical fidelity, validated against SPM12, and extended with neural amortized inference via normalizing flows.

## Milestone Summary

**Key Decisions:**
- Pyro over sbi/BayesFlow (need explicit generative model for ELBO comparison)
- torchdiffeq for ODE integration (PyTorch native, adjoint method)
- Zuko for normalizing flows (Pyro-compatible, clean API)
- Bilinear neural model first (Nozari 2024: linear suffices for macroscopic BOLD)
- SPM12 code defaults for hemodynamic parameters (not paper values)
- rk4 fixed-step for SVI (predictable runtime during optimization)
- Welch CSD for empirical computation (SPM matching via predicted CSD model)
- rDCM amortized guide skipped (analytic VB is exact for conjugate model)

**Issues Resolved:**
- ODE stiffness managed via log-space clamping and NaN detach protection
- CSD normalization exactly matches SPM conventions (C=1/256)
- Windows numpy process abort avoided via manual Pearson correlation
- Mean-field SVI coverage calibrated with empirically adjusted thresholds

**Technical Debt:**
- Amortization gap uses RMSE-ratio proxy (true ELBO gap needs wrapper model)
- rigid/sparse_inversion not in top-level __init__.py
- scripts/ missing __init__.py (namespace packages work but fragile)
- 1 flaky slow test (ODE stiffness on extended multi-seed runs)
