## v1 Requirements

### Infrastructure & Forward Models

- [x] **FWD-01**: Balloon-Windkessel hemodynamic model — Stephan et al. (2007) Eq. 2-5
- [x] **FWD-02**: Neural state equation dx/dt = Ax + Cu — Friston et al. (2003) Eq. 1
- [x] **FWD-03**: BOLD signal equation y = V0[k1(1-q) + k2(1-q/v) + k3(1-v)]
- [x] **FWD-04**: ODE integrator wrapper (torchdiffeq) with Euler, RK4, Dopri5
- [x] **FWD-05**: Cross-spectral density computation (Welch periodogram)
- [x] **FWD-06**: Spectral DCM transfer function H(w) with noise models
- [x] **FWD-07**: Regression DCM analytic frequency-domain likelihood

### Probabilistic Framework

- [x] **PROB-01**: Pyro generative model for task-based DCM
- [x] **PROB-02**: Pyro generative model for spectral DCM
- [x] **PROB-03**: Pyro generative model for regression DCM
- [x] **PROB-04**: Mean-field Gaussian variational guides

### Simulation & Testing

- [x] **SIM-01**: Task-based DCM simulator
- [x] **SIM-02**: Spectral DCM simulator
- [x] **SIM-03**: Regression DCM simulator
- [x] **REC-01**: Task DCM parameter recovery (RMSE, coverage, correlation)
- [x] **REC-02**: Spectral DCM parameter recovery
- [x] **REC-03**: Regression DCM parameter recovery
- [x] **REC-04**: ELBO convergence validation

### Validation

- [x] **VAL-01**: Task DCM cross-validated against SPM12 (auto-skip without MATLAB)

- [x] **AMR-02**: Amortized flow guide for spectral DCM
