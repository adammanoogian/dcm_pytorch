  <action>
    Implement the spectral transfer function and predicted CSD assembly. Use `from __future__ import annotations`, NumPy-style docstrings citing [REF-010], and type hints on all public functions. All operations use torch.complex128.

    **`default_frequency_grid(TR=2.0, n_freqs=32) -> torch.Tensor`:**
    - Returns frequency vector in Hz, shape (n_freqs,), float64.
    - SPM convention: lower bound = 1/128 Hz, upper bound = 1/(2*TR) Hz.
    - `torch.linspace(1.0/128.0, 1.0/(2.0*TR), n_freqs, dtype=torch.float64)`.
    - Cite: SPM12 spm_dcm_fmri_csd_data.m.

    **`compute_transfer_function(A, C_in, C_out, freqs) -> torch.Tensor`:**
    - Implements [REF-010] Eq. 3: g(w) = C_out @ (iwI - A)^-1 @ C_in via eigendecomposition.
    - A: (N, N) float64, effective connectivity (neural-only A for this phase).
    - C_in: (N, nu) float64, input projection (default: identity).
    - C_out: (nn, N) float64, output projection (default: identity).
    - freqs: (F,) float64, frequencies in Hz.
    - Returns: H, shape (F, nn, nu), complex128.
    - Step 1: Eigendecompose A: `eigvals, eigvecs = torch.linalg.eig(A.to(torch.complex128))`.
    - Step 2: Stabilize eigenvalues: `eigvals = torch.complex(torch.clamp(eigvals.real, max=-1.0/32.0), eigvals.imag)`. This is the SPM convention for fMRI frequencies. Cite: SPM12 spm_dcm_mtf.m.
    - Step 3: Project through eigenvectors: `dgdv = C_out.to(complex128) @ eigvecs`, `dvdu = torch.linalg.inv(eigvecs) @ C_in.to(complex128)`.
    - Step 4: Modal transfer: `Sk = 1.0 / (1j * 2 * pi * w[:, None] - eigvals[None, :])`, shape (F, N).
    - Step 5: Assemble: `H = torch.einsum('ik,kj,fk->fij', dgdv, dvdu, Sk)`, shape (F, nn, nu).
    - Cite [REF-010] Eq. 3 and SPM12 spm_dcm_mtf.m.

    **`predicted_csd(H, Gu, Gn) -> torch.Tensor`:**
    - Implements [REF-010] Eq. 4: S(w) = H(w) @ Gu(w) @ H(w)^H + Gn(w).
    - H: (F, nn, nu) complex128.
    - Gu: (F, nu, nu) complex128, neuronal noise CSD.
    - Gn: (F, nn, nn) complex128, observation noise CSD.
    - Returns: (F, nn, nn) complex128.
    - Formula: `G = H @ Gu @ H.conj().transpose(-2, -1) + Gn`.
    - Cite [REF-010] Eq. 4 and SPM12 spm_csd_fmri_mtf.m.

    **`spectral_dcm_forward(A, freqs, a, b, c) -> torch.Tensor`:**
    - Convenience function wrapping the full predicted CSD pipeline.
    - A: (N, N) float64, freqs: (F,) float64.
    - a: (2, N) float64, neuronal noise params (from spectral_noise module).
    - b: (2, 1) float64, global observation noise params.
    - c: (2, N) float64, regional observation noise params.
    - Returns: predicted CSD, (F, N, N) complex128.
    - Internally: computes H with C_in=C_out=identity (SPM convention for standard spDCM), builds noise spectra using spectral_noise module functions, assembles via predicted_csd.
    - Cite [REF-010] Eq. 3-7.

    Implement the SPM12 noise spectral models. Use `from __future__ import annotations`, NumPy-style docstrings citing [REF-010], and type hints. All outputs are complex128.

    **`neuronal_noise_csd(freqs, a) -> torch.Tensor`:**
    - Implements [REF-010] Eq. 5-6: Neuronal fluctuation CSD (diagonal).
    - freqs: (F,) float64, a: (2, N) float64 where a[0,:] = log amplitude, a[1,:] = log exponent.
    - Returns: Gu, shape (F, N, N) complex128, diagonal.
    - Formula per region i: `G_i(w) = C * exp(a[0,i]) * w^(-exp(a[1,i])) * 4.0` where C = 1/256 is SPM's scaling constant.
    - Implementation: Vectorize over regions. `amp = torch.exp(a[0, :])`, `exp_val = torch.exp(a[1, :])`. `G = (1.0/256.0) * amp[None, :] * freqs[:, None]**(-exp_val[None, :]) * 4.0`, shape (F, N).
    - Pack into diagonal: use `torch.diag_embed` or explicit indexing to create (F, N, N) diagonal complex128 matrix.
    - Cite [REF-010] Eq. 5-6 and SPM12 spm_csd_fmri_mtf.m.

    **`observation_noise_csd(freqs, b, c, n_regions) -> torch.Tensor`:**
    - Implements [REF-010] Eq. 7: Observation noise CSD (global + regional).
    - freqs: (F,) float64, b: (2, 1) float64, c: (2, N) float64, n_regions: int.
    - Returns: Gn, shape (F, N, N) complex128.
    - Global component (applies to all pairs i,j): `G_global(w) = C * exp(b[0,0]) * w^(-exp(b[1,0])/2) / 8.0`. This creates an (F,) vector broadcast to all entries of the (F, N, N) matrix. SPM divides exponent by 2 for observation noise (flatter spectrum than neuronal). The /8.0 factor matches SPM's spm_csd_fmri_mtf.m.
    - Regional component (diagonal only, adds to global): `G_regional_i(w) = C * exp(c[0,i]) * w^(-exp(c[1,i])/2)`. Adds to diagonal entries Gn[:, i, i].
    - C = 1/256 throughout.
    - Cite [REF-010] Eq. 7 and SPM12 spm_csd_fmri_mtf.m.

    **`default_noise_priors(n_regions) -> dict`:**
    - Returns dict with SPM12 default prior expectations and variances.
    - Keys: `a_prior_mean` (2, N) zeros, `a_prior_var` (2, N) filled with 1/64, `b_prior_mean` (2, 1) zeros, `b_prior_var` (2, 1) filled with 1/64, `c_prior_mean` (2, N) zeros, `c_prior_var` (2, N) filled with 1/64.
    - All float64. Total parameters: 4N + 2 (matching SPM12 spm_dcm_fmri_priors.m).
    - Cite SPM12 spm_dcm_fmri_priors.m.
