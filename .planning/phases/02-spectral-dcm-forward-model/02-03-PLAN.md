  <action>
    Implement the spectral DCM simulator. Use `from __future__ import annotations`, NumPy-style docstrings citing [REF-010], and type hints. Follow the pattern established by task_simulator.py.

    **`simulate_spectral_dcm(A, noise_params=None, TR=2.0, n_freqs=32, seed=None) -> dict`:**
    - Generates synthetic CSD from the spectral DCM model.
    - A: torch.Tensor (N, N), effective connectivity. Must have all eigenvalues with negative real parts (checked with assertion).
    - noise_params: dict or None. If None, use default priors (zeros for all from default_noise_priors). Keys: 'a' (2, N), 'b' (2, 1), 'c' (2, N) — all torch.Tensor float64.
    - TR: float, repetition time in seconds.
    - n_freqs: int, number of frequency bins.
    - seed: int or None, for reproducibility (only relevant if noise sampling is added).
    - Returns dict with:
      - `'csd'`: predicted CSD, shape (F, N, N) complex128.
      - `'freqs'`: frequency vector, shape (F,) float64.
      - `'transfer_function'`: H, shape (F, N, N) complex128.
      - `'neuronal_noise'`: Gu, shape (F, N, N) complex128.
      - `'observation_noise'`: Gn, shape (F, N, N) complex128.
      - `'params'`: dict with A, noise_params, TR, n_freqs.
    - Implementation: calls default_frequency_grid(TR, n_freqs) for freqs, then spectral_dcm_forward(A, freqs, a, b, c). Also computes and stores individual components (H, Gu, Gn) by calling compute_transfer_function and noise functions directly.
    - Cite [REF-010] and SIM-02 requirement.

    **`make_stable_A_spectral(n_regions, connection_strength=0.1, self_connection=-0.5, seed=None) -> torch.Tensor`:**
    - Convenience function to generate a stable A matrix suitable for spectral DCM simulation.
    - Similar to make_random_stable_A from Phase 1 but with spectral DCM conventions.
    - n_regions: int, self_connection: float (negative), connection_strength: float (scale of off-diagonal).
    - Returns: A (N, N) float64 with guaranteed negative eigenvalues.
    - Implementation: diagonal = self_connection, off-diagonal = connection_strength * randn, then verify eigenvalues and rescale if needed.
    - If seed is provided, set torch manual seed for reproducibility.
  </action>
