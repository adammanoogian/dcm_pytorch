    Implement empirical CSD computation from BOLD time series. Use `from __future__ import annotations`, NumPy-style docstrings, and type hints. This module uses numpy/scipy for computation (data preparation, not the differentiable forward model).

    **`compute_empirical_csd(bold, fs, freqs, nperseg=None) -> np.ndarray`:**
    - Computes cross-spectral density matrix from multivariate BOLD time series using scipy.signal.csd (Welch method).
    - bold: np.ndarray, shape (T, N) — T time points, N regions.
    - fs: float — sampling frequency in Hz (= 1/TR).
    - freqs: np.ndarray, shape (F,) — target frequency grid in Hz to interpolate onto.
    - nperseg: int or None — segment length for Welch. Default: min(256, T) to handle short time series.
    - Returns: complex CSD matrix, shape (F, N, N).
    - Implementation:
      1. For each region pair (i, j), compute `f_raw, Pxy = scipy.signal.csd(bold[:, i], bold[:, j], fs=fs, nperseg=nperseg, scaling='density')`.
      2. Interpolate real and imaginary parts separately onto the target frequency grid using `np.interp(freqs, f_raw, Pxy.real)` and `np.interp(freqs, f_raw, Pxy.imag)`.
      3. Assemble into (F, N, N) complex array.
    - The output should be Hermitian: `csd[:, i, j] = conj(csd[:, j, i])`. Enforce this by computing only upper triangle and conjugating.
    - Cite: FWD-05 requirement, scipy.signal.csd documentation.

    **`bold_to_csd_torch(bold, fs, freqs, nperseg=None) -> torch.Tensor`:**
    - Convenience wrapper that accepts torch.Tensor bold, converts to numpy, calls compute_empirical_csd, and returns torch.Tensor complex128.
    - bold: torch.Tensor (T, N), fs: float, freqs: torch.Tensor (F,).
    - Returns: torch.Tensor (F, N, N) complex128.
    - This is the interface that Phase 4's Pyro model will use to prepare observed data.

    **`default_welch_params(T, fs) -> dict`:**
    - Returns sensible Welch parameters for fMRI data.
    - nperseg: min(256, T) — Welch segment length.
    - noverlap: nperseg // 2 — 50% overlap (scipy default).
    - Returns dict with keys: nperseg, noverlap, window ('hann').
    - This is a helper for users who want to inspect/override defaults.
