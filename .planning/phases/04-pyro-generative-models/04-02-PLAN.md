    **Helper function `decompose_csd_for_likelihood(csd_complex: torch.Tensor) -> torch.Tensor`:**
    - Convert complex CSD tensor (F, N, N) to a real-valued vector for use in Pyro likelihood.
    - Flatten to 1D, then stack real and imaginary parts: `torch.cat([csd_complex.real.reshape(-1), csd_complex.imag.reshape(-1)])`.
    - Returns float64 tensor of shape `(2 * F * N * N,)`.
    - This is REQUIRED because Pyro's `dist.Normal` does not support complex128 tensors.
    - Cite: 04-RESEARCH.md Pattern 2.

    **`spectral_dcm_model(observed_csd, freqs, a_mask, N=None)`:**
    - This is a Pyro model function for spectral DCM.
    - Parameters:
      - `observed_csd`: (F, N, N) complex128 tensor, observed cross-spectral density.
      - `freqs`: (F,) float64 tensor, frequency vector in Hz.
      - `a_mask`: (N, N) float64 binary tensor, 1 where A connection exists.
      - `N`: int or None. If None, infer from a_mask.shape[0].
    - Implementation:
      1. Infer N from a_mask if not provided. Extract F from freqs.
      2. **Sample A_free** from `dist.Normal(zeros(N,N, dtype=float64), (1/64)**0.5 * ones(N,N, dtype=float64)).to_event(2)`. Apply mask: `A_free = A_free * a_mask`. Apply transform: `A = pyro.deterministic("A", parameterize_A(A_free))`.
      3. **Sample noise parameters** matching SPM12 spm_dcm_fmri_priors.m conventions:
         - `noise_a = pyro.sample("noise_a", dist.Normal(zeros(2,N, dtype=float64), (1/64)**0.5 * ones(2,N, dtype=float64)).to_event(2))` -- neuronal noise (log amplitude, log exponent) per region.
         - `noise_b = pyro.sample("noise_b", dist.Normal(zeros(2,1, dtype=float64), (1/64)**0.5 * ones(2,1, dtype=float64)).to_event(2))` -- global observation noise.
         - `noise_c = pyro.sample("noise_c", dist.Normal(zeros(2,N, dtype=float64), (1/64)**0.5 * ones(2,N, dtype=float64)).to_event(2))` -- regional observation noise.
      4. **Forward model:** Call `predicted_csd_complex = spectral_dcm_forward(A, freqs, noise_a, noise_b, noise_c)`. Register: `pyro.deterministic("predicted_csd", predicted_csd_complex)`.
      5. **Decompose complex to real:** `pred_real = decompose_csd_for_likelihood(predicted_csd_complex)`. `obs_real = decompose_csd_for_likelihood(observed_csd)`.
      6. **CSD observation noise:** Sample `csd_noise_scale = pyro.sample("csd_noise_scale", dist.HalfCauchy(tensor(1.0, dtype=float64)))`. This accounts for model-data mismatch beyond the generative noise model.
      7. **Likelihood:** `pyro.sample("obs_csd", dist.Normal(pred_real, csd_noise_scale).to_event(1), obs=obs_real)`. Using `.to_event(1)` treats the entire flattened CSD vector as a single observation.

    IMPORTANT anti-patterns to avoid:
    - Do NOT pass complex128 tensors to `pyro.sample` or `dist.Normal`. Always decompose first.
    - Do NOT sample C matrix -- spectral DCM uses C_in = C_out = identity internally (the spectral_dcm_forward function handles this).
    - Do NOT sample hemodynamic parameters -- spectral DCM operates entirely in frequency domain.
    - All real-valued tensors must be float64.
    - The predicted CSD from `spectral_dcm_forward` is complex128. Only decompose for the likelihood, NOT for the deterministic site (store the original complex tensor as the deterministic for downstream analysis).
  </action>
