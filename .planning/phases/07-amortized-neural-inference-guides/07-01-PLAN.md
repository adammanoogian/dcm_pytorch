**BoldSummaryNet(nn.Module):**
- Constructor args: `n_regions: int`, `embed_dim: int = 128`
- Architecture (from 07-RESEARCH.md):
  - `Conv1d(n_regions, 64, kernel_size=5, padding=2)` + `BatchNorm1d(64)` + `ReLU`
  - `Conv1d(64, 128, kernel_size=5, padding=2)` + `BatchNorm1d(128)` + `ReLU`
  - `Conv1d(128, 256, kernel_size=5, padding=2)` + `BatchNorm1d(256)` + `ReLU`
  - `AdaptiveAvgPool1d(1)` to squeeze temporal dimension
  - `Linear(256, embed_dim)`
- `forward(self, bold: torch.Tensor) -> torch.Tensor`:
  - Input: `bold` shape `(T, N)` or `(batch, T, N)` (handle both)
  - If no batch dim, unsqueeze at dim 0
  - Transpose last two dims to get `(batch, N, T)` for Conv1d channels-first
  - Pass through conv layers + pool + linear
  - Output: `(batch, embed_dim)` or `(embed_dim,)` if input had no batch dim
  - All float64 (project convention)

**CsdSummaryNet(nn.Module):**
- Constructor args: `n_regions: int`, `n_freqs: int = 32`, `embed_dim: int = 128`
- Architecture (from 07-RESEARCH.md):
  - Input CSD is complex `(F, N, N)` -- decompose to real/imag, flatten to `(2*F*N*N,)`
  - `Linear(2*F*N*N, 512)` + `ReLU`
  - `Linear(512, 256)` + `ReLU`
  - `Linear(256, embed_dim)`
- `forward(self, csd: torch.Tensor) -> torch.Tensor`:
  - Input: complex CSD shape `(F, N, N)` or `(batch, F, N, N)`
  - If no batch dim, unsqueeze at dim 0
  - Decompose to real/imag, flatten to `(batch, 2*F*N*N)`
  - Pass through MLP
  - Output: `(batch, embed_dim)` or `(embed_dim,)` if unbatched
  - MLP operates on float64

Both classes: NumPy-style docstrings, `from __future__ import annotations`, type hints, cite 07-RESEARCH.md and REF-042/REF-043 for amortized inference context.

1. Create `src/pyro_dcm/guides/parameter_packing.py` with two packer classes:

**TaskDCMPacker:**
- Constructor: `n_regions: int`, `n_inputs: int`, `a_mask: torch.Tensor`, `c_mask: torch.Tensor`
- Computes `n_features` = n_regions*n_regions + n_regions*n_inputs + 1 (A_free + C + noise_prec)
- NOTE: `task_dcm_model` has exactly three sample sites: `A_free` (N,N), `C` (N,M), `noise_prec` (scalar Gamma). Hemodynamic parameters are FIXED at SPM12 defaults and are NOT sampled, so they are NOT packed.
- `pack(params: dict[str, torch.Tensor]) -> torch.Tensor`:
  - Takes dict with keys `"A_free"` (N,N), `"C"` (N,M), `"noise_prec"` (scalar)
  - Flattens and concatenates into single vector of shape `(n_features,)`
  - Order: A_free.flatten(), C.flatten(), noise_prec (as 1-element)
- `unpack(z: torch.Tensor) -> dict[str, torch.Tensor]`:
  - Takes vector of shape `(..., n_features)` (supports batch dims)
  - Splits and reshapes back to named dict
  - Returns dict with `"A_free"` (N,N), `"C"` (N,M), `"noise_prec"` (scalar)
- `fit_standardization(dataset: list[dict]) -> None`:
  - Takes list of param dicts from training data
  - Packs all, computes per-element mean and std
  - Stores `self.mean_` and `self.std_` (trailing underscore per coding conventions)
  - Clamps std to min 1e-6 to avoid division by zero
- `standardize(z: torch.Tensor) -> torch.Tensor`:
  - `(z - self.mean_) / self.std_` -- critical for NSF spline domain [-5, 5]
- `unstandardize(z_std: torch.Tensor) -> torch.Tensor`:
  - `z_std * self.std_ + self.mean_`

**SpectralDCMPacker:**
- Constructor: `n_regions: int`
- `n_features` = n_regions*n_regions + 2*n_regions + 2 + 2*n_regions + 1 (A_free + noise_a + noise_b + noise_c + csd_noise_scale)
- NOTE: `spectral_dcm_model` sample sites are: `A_free` (N,N), `noise_a` (2,N), `noise_b` (2,1), `noise_c` (2,N), `csd_noise_scale` (scalar HalfCauchy). These names must match exactly.
- Same pack/unpack/standardize interface but for spectral DCM sites:
  - `"A_free"` (N,N), `"noise_a"` (2,N), `"noise_b"` (2,1), `"noise_c"` (2,N), `"csd_noise_scale"` (scalar)

Both classes: store `a_mask`/`c_mask` for applying structural masking during unpack. Docstrings cite 07-RESEARCH.md Pitfall 3 (spline domain truncation) for standardization rationale.

2. Update `src/pyro_dcm/guides/__init__.py` to also export `TaskDCMPacker`, `SpectralDCMPacker`.

3. Create `scripts/generate_training_data.py`:
- Argparse CLI with args: `--variant` (task/spectral), `--n-simulations` (default 10000), `--n-regions` (default 3), `--n-inputs` (default 1), `--output-dir` (default `data/training/`), `--seed` (default 42)
- For task DCM:
  - Generate random stable A via `make_random_stable_A(n_regions, seed=seed+i)`
  - Random C from N(0, 1) masked by ones(N, M)
  - Create block stimulus via `make_block_stimulus(n_blocks=5, block_duration=30, rest_duration=30)`
  - Call `simulate_task_dcm(A, C, stimulus, TR=2.0, SNR=5.0)`
  - Extract `A_free` from A by inverting `parameterize_A`:
    - Off-diagonal: `A_free_ij = A_ij` (identity, no transform)
    - Diagonal: `A_free_ii = log(-2 * A_ii)` (inverting `A_ii = -exp(x)/2`)
    - Verified from `neural_state.py` line 44: `A[diag_mask] = -torch.exp(A_free[diag_mask]) / 2.0`
  - Store `{"bold": result["bold"], "A_free": A_free, "C": C, "noise_prec": torch.tensor(SNR**2 / signal_var)}`
  - Filter out simulations with NaN/Inf bold (07-RESEARCH.md Pitfall 5)
  - Progress bar (print every 100 simulations)
- For spectral DCM:
  - Generate random stable A via `make_stable_A_spectral(n_regions, seed=seed+i)`
  - Use `default_noise_priors` for noise params
  - Call `simulate_spectral_dcm(A, seed=seed+i)`
  - Extract A_free from A using the same inversion: off-diag identity, diag `log(-2 * A_ii)`
  - Store `{"csd": result["csd"], "A_free": A_free, "noise_a": ..., "noise_b": ..., "noise_c": ..., "csd_noise_scale": ...}`
  - NOTE: `simulate_spectral_dcm` returns `result["csd"]` (not `result["predicted_csd"]`)
- Save dataset as a single .pt file: `torch.save({"data": [...], "params": [...]}, path)`
- Also compute and save standardization stats (mean, std of packed params)

This validates the inversion formula is correct.
