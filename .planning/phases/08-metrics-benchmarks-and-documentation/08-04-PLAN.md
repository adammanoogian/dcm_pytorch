<task type="auto">
  <name>Task 1: Create plotting module</name>
  <files>
    benchmarks/plotting.py
  </files>
  <action>
Create benchmarks/plotting.py with figure-generation functions. Each function takes
a results dict (loaded from JSON) and an output directory, produces one figure.

Style handling:
- Try `import scienceplots; plt.style.use(['science', 'no-latex'])` first.
  If SciencePlots not installed, fall back to
  `plt.style.use('seaborn-v0_8-whitegrid')`.
  If that fails too, use default matplotlib style.
  Wrap in try/except, log a warning (print) if falling back.
- All figures: figsize=(8, 5), dpi=150, tight_layout.
- Save both .pdf and .png: `fig.savefig(path + '.pdf')` and
  `fig.savefig(path + '.png', dpi=150)`.
- Use colorblind-friendly palette: tab10 or Set2.

Functions:

1. `plot_rmse_comparison(results: dict, output_dir: str) -> None`
   Grouped bar chart: x-axis = DCM variants (Task, Spectral, rDCM-rigid, rDCM-sparse),
   bars grouped by method (SVI, Amortized, Analytic VB).
   y-axis = mean RMSE(A) with error bars (std).
   Title: "Parameter Recovery: RMSE(A) by Variant and Method"

2. `plot_time_comparison(results: dict, output_dir: str) -> None`
   Grouped bar chart: same layout as RMSE but y-axis = wall time (seconds).
   Use log scale if max/min ratio > 100.
   Title: "Inference Time per Subject"

3. `plot_coverage_comparison(results: dict, output_dir: str) -> None`
   Grouped bar chart: y-axis = 90% CI coverage.
   Add horizontal line at 0.90 (nominal level).
   Title: "Posterior Calibration: 90% CI Coverage"

4. `plot_amortization_gap(results: dict, output_dir: str) -> None`
   Bar chart: x-axis = variants that have amortized results (Task, Spectral),
   y-axis = relative amortization gap (%).
   Add horizontal line at 10% (target threshold).
   Title: "Amortization Gap: ELBO Degradation from Amortization"

5. `plot_true_vs_inferred(results: dict, output_dir: str) -> None`
   Scatter plot: x = true A elements, y = inferred A elements.
   Different colors per variant. Add identity line (y=x).
   Show Pearson correlation in legend.
   Title: "True vs. Inferred Connectivity (A matrix)"

6. `generate_all_figures(results: dict, output_dir: str = "figures") -> None`
   Calls all 5 plotting functions. Creates output_dir if needed.
   Prints "Generated {n} figures in {output_dir}/".

7. `load_results(path: str = "benchmarks/results/benchmark_results.json") -> dict`
   Convenience function to load JSON results.

Add a `if __name__ == "__main__":` block that loads results and generates all figures.

Handle missing data gracefully: if a variant/method is not in results, skip that
bar/point. If results file doesn't exist, print error and exit.
  </action>
