**docs/03_methods_reference/methods.md** -- Primary Markdown version:

Section structure:

1. **Generative Model**
   - 1.1 Neural State Equation: dx/dt = Ax + Cu [REF-001] Eq. 1.
     Describe bilinear model, A matrix interpretation, C matrix, state vector.
   - 1.2 Hemodynamic Model: Balloon-Windkessel [REF-002] Eq. 2-5.
     Four coupled ODEs (s, f, v, q). Parameter table with priors (kappa, gamma,
     tau, alpha, E0 -- use SPM12 defaults from STATE.md).
   - 1.3 BOLD Signal: y = V0[k1(1-q) + k2(1-q/v) + k3(1-v)] [REF-002] Eq. 6.
     Constants k1, k2, k3 with simplified Buxton form.
   - 1.4 Spectral DCM: Transfer function H(w) = (iwI - A)^-1 [REF-010] Eq. 3.
     Predicted CSD, neuronal noise (1/f^alpha), observation noise model.
   - 1.5 Regression DCM: Frequency-domain regression [REF-020] Eq. 4-8.
     HRF convolution in frequency domain, design matrix construction.

2. **Inference**
   - 2.1 Stochastic Variational Inference: ELBO objective, mean-field Gaussian
     guide (AutoNormal), Adam optimizer, learning rate schedule.
   - 2.2 Amortized Inference: Summary networks (1D-CNN for BOLD, MLP for CSD),
     Neural Spline Flow (Zuko NSF), ZukoToPyro wrapper, parameter standardization.
   - 2.3 Analytic VB for rDCM: Closed-form posterior [REF-020] Eq. 11-14,
     free energy [REF-020] Eq. 15, ARD sparsity priors.

3. **Bayesian Model Comparison**
   - ELBO-based comparison for task/spectral DCM.
   - Free energy comparison for rDCM.
   - Interpretation (higher ELBO/F = better model).

4. **Implementation**
   - Software stack: PyTorch 2.x, Pyro 1.9+, torchdiffeq, Zuko, scipy.
   - Numerical stability: log-space parameters, eigenvalue clamping, NaN protection.
   - ODE integration: rk4 fixed-step for SVI, configurable solver for simulation.

5. **Benchmarks** (placeholder for results from Plan 08-04)
   - Protocol description: simulate -> infer -> compare.
   - Metrics: RMSE, coverage, correlation, ELBO, wall time.
   - Amortization gap definition.
   - Table placeholder: "See benchmark_report.md for results."
