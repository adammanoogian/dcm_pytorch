# Phase 3 Plan 01: rDCM Forward Pipeline Summary

**One-liner:** Euler-integrated HRF, zero-padded FFT BOLD generation, and frequency-domain design matrix construction with derivative coefficients and real/imaginary splitting following Julia RegressionDynamicCausalModeling.jl.

## What Was Built

### rdcm_forward.py (8 functions)

1. **dcm_euler_step** -- Single Euler step for a multi-region DCM with neural and hemodynamic dynamics. Uses Julia convention with clamping for numerical stability.

2. **euler_integrate_dcm** -- Full Euler integration loop producing neural activity and BOLD signal at each time step. BOLD computed with Julia's relaxationRateSlope/frequencyOffset constants.

3. **get_hrf** -- Generates HRF by Euler-integrating a minimal 1-region DCM (A=-1, C=16) with unit impulse input. Peaks at ~6.1s matching canonical HRF shape.

4. **generate_bold** -- Complete synthetic BOLD pipeline: Euler-integrate DCM, 3x zero-pad for non-circular convolution, FFT convolve with HRF, subsample to TR, add Gaussian noise scaled by per-region SNR.

5. **compute_derivative_coefficients** -- DFT derivative coefficients coef[k] = exp(2*pi*i*k/N) - 1 for transforming BOLD DFT to temporal-derivative DFT.

6. **split_real_imag** -- Splits complex rfft output into stacked real/imaginary parts, excluding zero-valued DC imaginary (always) and Nyquist imaginary (even N only).

7. **reduce_zeros** -- Balances zero vs non-zero frequency rows by replacing excess zero rows with NaN markers.

8. **create_regressors** -- Full design matrix construction: FFT convolution of stimulus with HRF, subsampling, derivative coefficients applied to Y, real/imag splitting, zero reduction. Output is float64 X and Y ready for Bayesian linear regression.

### test_rdcm_forward.py (27 tests)

Comprehensive coverage across all 8 functions:
- Euler integration: equilibrium stability, impulse response, shape, 500s stability
- HRF: shape, peak timing, positivity, baseline return, no NaN
- BOLD generation: shape, SNR control, dict keys, no NaN, zero-padding verification
- Derivative coefficients: shape, DC=0, analytic formula match
- Real/imag splitting: even N, odd N, DC exclusion, Nyquist exclusion
- reduce_zeros: no-op and NaN balancing
- create_regressors: shape, dtype, valid rows, derivative presence in Y

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| Euler-integrated HRF (not double-gamma) | Matches Julia exactly; different shape than parametric HRF |
| Julia BOLD constants (k1=0.4, k2=0.644, k3=1.0, V0=4.0) | Different from Phase 1 SPM BOLD; rDCM has its own convention |
| H[5]=rho=0.32 for oxygen extraction in Euler | Julia dcm_euler_integration.jl convention |
| 3x zero-padding for FFT convolution | Avoids circular artifacts; matches Julia |
| NaN markers for zero-frequency balancing | Julia reduce_zeros! convention |
