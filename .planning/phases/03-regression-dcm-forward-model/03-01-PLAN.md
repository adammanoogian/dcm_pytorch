  <action>
    Implement the rDCM forward pipeline module. Use `from __future__ import annotations`, NumPy-style docstrings citing [REF-020], type hints on all public functions. All real operations use torch.float64, all frequency-domain operations use torch.complex128.

    **`dcm_euler_step(x, s, f, v, q, A, C, u_t, dt, H) -> tuple`:**
    - Single Euler step for a DCM with nr regions. Implements Julia `dcm_euler_integration.jl`.
    - x: (nr,) neural activity, s: (nr,) vasodilatory signal, f: (nr,) blood flow, v: (nr,) venous volume, q: (nr,) deoxyhemoglobin.
    - A: (nr, nr) connectivity, C: (nr, nu) input weights, u_t: (nu,) input at time t.
    - dt: float, time step. H: tuple of 5 hemodynamic constants (kappa, gamma, tau, alpha, rho).
    - Neural: `x_new = x + dt * (A @ x + C @ u_t)`.
    - Hemodynamic (per region, sequential update matching Julia):
      - `s_new = s + dt * (x - H[0]*s - H[1]*(f - 1))`
      - `f_new = f + dt * s`
      - `v_new = v + dt * (f - v^(1/H[3])) / H[2]`
      - `q_new = q + dt * (f * (1-(1-H[4])^(1/f)) / H[4] - v^(1/H[3]) * q/v) / H[2]`
    - Clamp f, v, q to min 1e-6 to prevent numerical issues.
    - Returns (x_new, s_new, f_new, v_new, q_new).
    - Cite: Julia dcm_euler_integration.jl.

    **`euler_integrate_dcm(A, C, u, dt, H) -> tuple[torch.Tensor, torch.Tensor]`:**
    - Full Euler integration of DCM over time.
    - u: (N_t, nu) stimulus input.
    - Returns: x (N_t, nr) neural activity, bold (N_t, nr) BOLD signal.
    - Initialize all states to equilibrium: x=0, s=0, f=1, v=1, q=1.
    - BOLD signal per step: Uses Julia BOLD formula with constants:
      - `relaxationRateSlope = 25.0`, `frequencyOffset = 40.3`, `oxygenExtractionFraction = 0.4`, `echoTime = 0.04`, `restingVenousVolume = 4.0`.
      - `k1 = relaxationRateSlope * oxygenExtractionFraction * echoTime`
      - `k2 = frequencyOffset * oxygenExtractionFraction * echoTime`
      - `k3 = 1.0`  (from Julia: 1 - epsilon)
      - `bold = restingVenousVolume * (k1*(1-q) + k2*(1-q/v) + k3*(1-v))`
    - Loop over all timesteps calling dcm_euler_step.
    - Cite: Julia dcm_euler_integration.jl, generate_BOLD.jl.

    **`get_hrf(N: int, u_dt: float) -> torch.Tensor`:**
    - Generate HRF by Euler-integrating a minimal 1-region DCM.
    - Creates A=[[-1.0]], C=[[16.0]] (Julia divides C by 16 internally, so C_eff = 1.0 — check Julia code and match exactly).
    - Unit impulse input: u[0] = 1.0, u[1:] = 0.0, length N.
    - Hemodynamic constants: H = (0.64, 0.32, 2.00, 0.32, 0.32).
    - Calls euler_integrate_dcm, returns the BOLD output as the HRF.
    - Cite: Julia generate_BOLD.jl get_hrf().

    **`generate_bold(A, C, u, u_dt, y_dt, SNR=1.0) -> dict`:**
    - Generate synthetic BOLD for rDCM simulation.
    - Steps (matching Julia generate_BOLD.jl):
      1. Compute HRF via get_hrf(N_u, u_dt).
      2. Euler-integrate DCM with A, C, u to get neural signal x (N_u, nr).
      3. Zero-pad x and hrf to 3*N_u to avoid circular convolution artifacts.
      4. Convolve x with hrf in frequency domain per region: `y = ifft(fft(x_padded) * fft(hrf_padded))`.
      5. Truncate back to N_u, subsample at ratio y_dt/u_dt to get BOLD at TR.
      6. Add Gaussian noise: `noise = randn * std(y_col) / SNR` per region.
    - Returns dict with keys: 'y' (N_y, nr), 'y_clean' (N_y, nr), 'x' (N_u, nr), 'hrf' (N_u,).
    - Cite: Julia generate_BOLD.jl generate_BOLD().

    **`compute_derivative_coefficients(N: int) -> torch.Tensor`:**
    - Compute DFT derivative coefficients.
    - `coef[k] = exp(2*pi*i*k/N) - 1` for k = 0, ..., N//2.
    - Returns complex128 tensor of shape (N//2 + 1,).
    - Cite: [REF-020] Eq. 6-7, Julia create_regressors.jl.

    **`split_real_imag(Z_complex: torch.Tensor, N_y: int) -> torch.Tensor`:**
    - Split complex frequency-domain data into stacked real and imaginary parts.
    - Z_complex: (N_rfft, ...) complex128 output of rfft.
    - If N_y is even: `Z_real = cat([Z.real, Z.imag[1:-1]], dim=0)` (exclude DC and Nyquist imag).
    - If N_y is odd: `Z_real = cat([Z.real, Z.imag[1:]], dim=0)` (exclude DC imag only).
    - Returns float64 tensor.
    - Cite: Julia rigid_inversion.jl, sparse_inversion.jl.

    **`reduce_zeros(Y: torch.Tensor, X: torch.Tensor) -> tuple[torch.Tensor, torch.Tensor]`:**
    - Balance zero vs non-zero frequency rows. Matching Julia `reduce_zeros!`.
    - For each region r, count zero-valued rows in Y[:, r]. If more zeros than non-zeros, set some rows to NaN to balance. This improves regression quality.
    - Returns modified Y, X with NaN markers for excluded rows.
    - Cite: Julia create_regressors.jl reduce_zeros!.

    **`create_regressors(hrf, y, u, u_dt, y_dt, X0=None) -> tuple[torch.Tensor, torch.Tensor, int]`:**
    - Create frequency-domain design matrix and data for rDCM regression.
    - hrf: (N_u,) HRF from get_hrf.
    - y: (N_y, nr) BOLD signal.
    - u: (N_u, nu) stimulus inputs.
    - u_dt: float, input sampling interval.
    - y_dt: float, BOLD sampling interval (TR).
    - X0: (N_u, nc) confound regressors, default None (adds constant confound).
    - Steps (matching Julia create_regressors.jl):
      1. If X0 is None, create constant confound: ones(N_u, 1).
      2. Compute FFT of HRF: `hrf_fft = rfft(hrf)`.
      3. Compute FFT of stimulus: `u_fft = rfft(u, dim=0)`.
      4. Convolve stimulus with HRF: `uh_fft = u_fft * hrf_fft[:, None]`, then `uh = irfft(uh_fft, dim=0)`.
      5. Subsample uh and confounds at ratio y_dt/u_dt.
      6. Compute derivative coefficients for N_y.
      7. Compute Y: `Y_complex = coef[:, None] * rfft(y, dim=0) / y_dt`.
      8. Compute X: `X_complex = cat([rfft(y, dim=0), rfft(uh_subsampled, dim=0)/y_dt, rfft(X0_subsampled, dim=0)], dim=1)`.
      9. Apply split_real_imag to both Y and X.
      10. Apply reduce_zeros.
    - Returns (X, Y, N_eff) where X: (N_eff, nr+nu+nc) float64, Y: (N_eff, nr) float64, N_eff: int effective data points.
    - Cite: [REF-020] Eq. 4-8, Julia create_regressors.jl.
