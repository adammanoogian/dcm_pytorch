  <name>Task 1: Prior specification, rigid VB inversion, and standalone likelihood — rdcm_posterior.py (Part A)</name>

  <action>
    Implement the rDCM prior specification, rigid VB inversion, and standalone likelihood function. Use `from __future__ import annotations`, NumPy-style docstrings citing [REF-020], type hints on all public functions. All operations use torch.float64. Note: `torch.special.digamma` requires PyTorch >= 1.9; this is already satisfied by the pyproject.toml `torch>=2.0` constraint.

    **`get_priors_rigid(a_mask, c_mask) -> dict`:**
    - Compute rDCM priors for rigid inversion. Matching Julia get_priors.jl.
    - a_mask: (nr, nr) binary architecture mask for A.
    - c_mask: (nr, nu) binary mask for C.
    - Prior mean [REF-020] Eq. 9:
      - A: `pE_A = zeros(nr, nr) - 0.5 * eye(nr)` (self-inhibition = -0.5, off-diagonal = 0).
      - C: `pE_C = zeros(nr, nu)`.
    - Prior covariance [REF-020] Eq. 10:
      - `a_off = a_mask.clone(); a_off.fill_diagonal_(0)` (off-diagonal mask).
      - `pC_A = (a_off * 8) / nr + eye(nr) / (8 * nr)`.
      - `pC_C = zeros(nr, nu); pC_C[c_mask.bool()] = 1.0`.
    - Prior precision: `l0_A = 1.0 / pC_A`, `l0_C = 1.0 / pC_C` (inf for absent connections, handled by architecture mask later).
    - Concatenate: `m0 = cat([pE_A, pE_C], dim=1)`, `l0 = cat([l0_A, l0_C], dim=1)`.
    - Noise precision: `a0 = 2.0`, `b0 = 1.0` (Gamma prior).
    - Returns dict with keys: 'm0', 'l0', 'a0', 'b0'.
    - Cite: Julia get_priors.jl, [REF-020] Eq. 9-10.

    **`get_priors_sparse(a_mask, c_mask) -> dict`:**
    - Compute rDCM priors for sparse inversion. Matching Julia get_priors.jl.
    - Key difference from rigid: uses `ones(nr, nr)` and `ones(nr, nu)` as architecture masks, since sparsity is learned via z indicators.
    - Prior mean: `pE_A.fill_diagonal_(-0.5)`, off-diagonal A mean = 0.
    - Prior covariance: computed with full connectivity masks.
    - Returns same dict structure as rigid plus 'p0' (Bernoulli prior, default 0.5).
    - Cite: Julia get_priors.jl.

    **`compute_rdcm_likelihood(Y_r, X_r, mu_r, tau_r) -> torch.Tensor`:**
    - Standalone rDCM analytic log-likelihood for one region.
    - This is the frequency-domain Gaussian likelihood from [REF-020] Eq. 15 (log_lik component), exposed as a standalone function for testing and downstream use.
    - Y_r: (N_eff_r,) data vector for region r (NaN-filtered).
    - X_r: (N_eff_r, D_r) design matrix for region r (NaN-filtered).
    - mu_r: (D_r,) posterior mean weights.
    - tau_r: float, posterior noise precision (a_r / beta_r).
    - Computes: `residual = Y_r - X_r @ mu_r`, `log_lik = -0.5 * N_eff_r * log(2*pi) + 0.5 * N_eff_r * log(tau_r) - 0.5 * tau_r * (residual @ residual)`.
    - Returns: scalar log-likelihood value.
    - Cite: [REF-020] Eq. 15 (log-likelihood component), Julia rigid_inversion.jl.

    **`compute_free_energy_rigid(N_eff, a_r, beta_r, QF, tau_r, l0_r, mu_r, mu0_r, Sigma_r, a0, beta0, D_r) -> float`:**
    - Compute negative free energy for one region [REF-020] Eq. 15.
    - Five additive components:
      - `log_lik = 0.5*(N_eff*(digamma(a_r) - log(beta_r)) - N_eff*log(2*pi)) - QF*tau_r`
      - `log_p_weight = 0.5*(slogdet(l0_r)[1] - D_r*log(2*pi) - (mu_r - mu0_r) @ l0_r @ (mu_r - mu0_r) - trace(l0_r @ Sigma_r))`
      - `log_p_prec = a0*log(beta0) - lgamma(a0) + (a0-1)*(digamma(a_r) - log(beta_r)) - beta0*tau_r`
      - `log_q_weight = 0.5*(slogdet(Sigma_r)[1] + D_r*(1 + log(2*pi)))`
      - `log_q_prec = a_r - log(beta_r) + lgamma(a_r) + (1-a_r)*digamma(a_r)`
    - Returns: F_r = log_lik + log_p_weight + log_p_prec + log_q_weight + log_q_prec.
    - Use `torch.special.digamma`, `torch.lgamma`, `torch.linalg.slogdet`.
    - Cite: [REF-020] Eq. 15, Julia rigid_inversion.jl compute_F().

    **`rigid_inversion(X, Y, a_mask, c_mask, confound_cols=1, max_iter=500, tol=1e-5) -> dict`:**
    - Region-wise VB inversion for rigid rDCM. Matching Julia rigid_inversion.jl.
    - X: (N_eff, nr+nu+nc) design matrix, Y: (N_eff, nr) data.
    - For each region r:
      1. Select columns from X based on architecture mask: `idx_r = cat([a_mask[r, :], c_mask[r, :], ones(nc)])`. Remove columns where idx_r is 0.
      2. Filter NaN rows: `valid = ~isnan(Y[:, r])`, apply to X_r and Y_r.
      3. Extract per-region prior: `mu0_r = m0[r, idx_r]`, `l0_r = diag(l0[r, idx_r])`.
      4. Precompute: `W = X_r.T @ X_r`, `V = X_r.T @ Y_r`.
      5. Initialize: `tau_r = a0/b0`, `a_r = a0 + N_eff_r * 0.5`, `F_old = -inf`, `pr = tol**2`.
      6. VB iteration loop (max_iter):
         - Posterior covariance [REF-020] Eq. 12: `Sigma_r = inv(tau_r * W + l0_r)`, symmetrize.
         - Posterior mean [REF-020] Eq. 11: `mu_r = Sigma_r @ (tau_r * V + l0_r @ mu0_r)`.
         - Quadratic form [REF-020] Eq. 14: `residual = Y_r - X_r @ mu_r`, `QF = 0.5 * (residual @ residual + trace(W @ Sigma_r))`.
         - Posterior Gamma rate: `beta_r = b0 + QF`, `tau_r = a_r / beta_r`.
         - Free energy [REF-020] Eq. 15: call compute_free_energy_rigid.
         - Convergence: `if (F_old - F_r)**2 < pr: break`.
         - Update: `F_old = F_r`.
      7. Store per-region: mu_r, Sigma_r, a_r, beta_r, F_r, iterations.
    - Assemble results: reshape mu values back into (nr, nr) A matrix and (nr, nu) C matrix.
    - Total free energy: `F_total = sum(F_r)`.
    - Returns dict with keys: 'A_mu' (nr, nr), 'C_mu' (nr, nu), 'mu_per_region' list, 'Sigma_per_region' list, 'a_per_region', 'beta_per_region', 'F_per_region', 'F_total', 'iterations_per_region'.
    - Cite: [REF-020] Eq. 11-15, Julia rigid_inversion.jl.
  </action>

  <action>
    Add sparse VB inversion to rdcm_posterior.py. This builds on the priors and rigid infrastructure from Task 1.

    **`compute_free_energy_sparse(N_eff, a_r, beta_r, QF, tau_r, l0_r, mu_r, mu0_r, Sigma_r, a0, beta0, D_r, z_r, z_idx, p0) -> float`:**
    - Sparse free energy with 7 components (5 from rigid + 2 for z indicators).
    - Additional components:
      - `log_p_z = sum(log(1 - p0[z_idx]) + z_r[z_idx] * log(p0[z_idx]/(1-p0[z_idx])))` where z_idx are indices where `tol**2 < z_r < 1`.
      - `log_q_z = sum(-(1-z[z_idx])*log(1-z[z_idx]+eps) - z[z_idx]*log(z[z_idx]+eps))` (entropy of z).
    - Returns: F_r = rigid_components + log_p_z + log_q_z.
    - Cite: Julia sparse_inversion.jl compute_F_sparse().

    **`sparse_inversion(X, Y, a_mask, c_mask, confound_cols=1, max_iter=500, tol=1e-5, n_reruns=100, p0=0.5, restrict_inputs=True) -> dict`:**
    - Sparse VB inversion with binary indicators. Matching Julia sparse_inversion.jl.
    - Key differences from rigid:
      1. Uses get_priors_sparse (all connections possible).
      2. Initializes z_r = 0.5 * ones(D_r) for A connections, z_r = 1.0 for C connections (if restrict_inputs=True).
      3. VB loop includes z update with random sweep ordering:
         - Compute `A_mat = W * (mu_r @ mu_r.T) + W * Sigma_r` (elementwise multiply).
         - For each i in `torch.randperm(D_r)`:
           - `g_i = log(p0/(1-p0)) + tau_r * mu_r[i] * V[i] + tau_r * A_mat[i,i] / 2`.
           - Set z_r[i] = 1 temporarily.
           - `g_i -= tau_r * z_r @ A_mat[:, i]`.
           - `z_r[i] = sigmoid(g_i)`.
         - If restrict_inputs: keep z=1 for C indices.
      4. Update Z = diag(z_r), G = Z @ W @ Z; G diagonal = z_r * W.diag().
      5. Posterior covariance: `Sigma_r = inv(tau_r * G + l0_r)`.
      6. Posterior mean: `mu_r = Sigma_r @ (tau_r * Z @ V + l0_r @ mu0_r)`.
      7. QF (sparse): `QF = 0.5*(Y^T @ Y - 2*mu^T @ Z @ V + mu^T @ G @ mu + trace(G @ Sigma))`.
      8. Hard thresholding: set `mu_r[abs(mu_r) < 1e-5] = 0`, set `z_r[mu_r == 0] = 0`.
    - Run n_reruns times, select the run with highest free energy.
    - Returns same structure as rigid_inversion plus 'z_per_region' list.
    - Cite: Julia sparse_inversion.jl, Frassle et al. (2018).
  </action>
