    **src/pyro_dcm/utils/ode_integrator.py:**

    `class PiecewiseConstantInput`:
    - `__init__(self, times: torch.Tensor, values: torch.Tensor)`: times shape (K,) sorted ascending, values shape (K, M) for M inputs.
    - `__call__(self, t: torch.Tensor) -> torch.Tensor`: Use `torch.searchsorted(self.times, t.detach(), right=True) - 1`, clamp to valid range, return `self.values[idx]`. Shape (M,).
    - `grid_points` property: Returns `self.times` tensor for solver restarts at discontinuities.

    `def integrate_ode(func, y0, t_eval, method='dopri5', rtol=1e-5, atol=1e-7, grid_points=None, adjoint=False)`:
    - Wrapper around `torchdiffeq.odeint` or `torchdiffeq.odeint_adjoint` (selected by `adjoint` flag).
    - Pass `options={'grid_points': grid_points, 'eps': 1e-6}` when grid_points is not None and method is adaptive (dopri5).
    - For fixed-step methods (euler, rk4), pass `options={'step_size': step_size}` where step_size is an optional parameter (default 0.01).
    - Returns solution tensor shape (T, D) where T=len(t_eval), D=state dimension.
    - Cite torchdiffeq API in docstring.

    `def make_initial_state(n_regions: int, dtype=torch.float64, device='cpu') -> torch.Tensor`:
    - Returns zeros tensor of shape (5 * n_regions,) — steady state initial conditions.
    - Document state layout: [x(N), s(N), lnf(N), lnv(N), lnq(N)].

    **src/pyro_dcm/forward_models/coupled_system.py:**

    `class CoupledDCMSystem(nn.Module)`:
    - `__init__(self, A, C, input_fn, hemo_params=None)`: Store A (N,N tensor), C (N,M tensor), input_fn (callable t->u), hemo_params (dict or use defaults). Instantiate NeuralStateEquation and BalloonWindkessel internally. A and C should be stored as buffers (register_buffer) NOT parameters, because the Pyro model will handle parameterization in Phase 4.
    - `forward(self, t, state) -> torch.Tensor`: This is the ODE right-hand side. Takes scalar t and state vector shape (5*N,). Returns derivative vector shape (5*N,).
      1. Unpack state: x = state[:N], s = state[N:2N], lnf = state[2N:3N], lnv = state[3N:4N], lnq = state[4N:5N].
      2. Get u(t) from input_fn.
      3. Compute dx = A @ x + C @ u (via NeuralStateEquation).
      4. Compute ds, dlnf, dlnv, dlnq = BalloonWindkessel.derivatives(x, s, lnf, lnv, lnq).
      5. Return torch.cat([dx, ds, dlnf, dlnv, dlnq]).
    - Cite [REF-001] Eq. 1 + [REF-002] Eq. 2-5.
