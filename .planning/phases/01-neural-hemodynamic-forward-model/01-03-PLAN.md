    `def simulate_task_dcm(A, C, stimulus, hemo_params=None, duration=300.0, dt=0.01, TR=2.0, SNR=5.0, solver='dopri5', device='cpu', dtype=torch.float64, seed=None) -> dict`:

    Parameters:
    - A: torch.Tensor (N, N) — effective connectivity (already parameterized, with negative diag). The function should accept raw A, NOT A_free. Document this.
    - C: torch.Tensor (N, M) — driving input weights.
    - stimulus: dict with 'times' (K,) and 'values' (K, M) for PiecewiseConstantInput. OR a PiecewiseConstantInput instance directly.
    - hemo_params: dict or None (use DEFAULT_HEMO_PARAMS from balloon_model.py).
    - duration: float, simulation duration in seconds.
    - dt: float, step size hint for fixed-step solvers.
    - TR: float, repetition time for BOLD downsampling (seconds).
    - SNR: float, signal-to-noise ratio. SNR = std(signal) / std(noise). If SNR <= 0, no noise added.
    - solver: str, one of 'dopri5', 'rk4', 'euler'.
    - device, dtype: for tensor creation.
    - seed: optional int for reproducibility.

    Implementation steps:
    1. Set random seed if provided (torch.manual_seed).
    2. Create PiecewiseConstantInput from stimulus dict (if not already).
    3. Create CoupledDCMSystem(A, C, input_fn, hemo_params).
    4. Create initial state via make_initial_state(N).
    5. Create t_eval: torch.arange(0, duration, dt) for fine-grained integration.
    6. Call integrate_ode with grid_points from input_fn.
    7. Extract states from solution: x, s, lnf, lnv, lnq — each shape (T_fine, N).
    8. Compute v = exp(lnv), q = exp(lnq).
    9. Compute clean_bold = bold_signal(v, q, E0, V0) — shape (T_fine, N).
    10. Downsample to TR: take every (TR/dt)-th sample. Use integer indexing: indices = torch.arange(0, T_fine, round(TR/dt)). Result shape (T_TR, N).
    11. Add Gaussian noise: noise_std = clean_bold_downsampled.std() / SNR per region. noisy_bold = clean_bold_downsampled + noise_std * randn.
    12. Return dict:
        - 'bold': noisy_bold (T_TR, N) — this is the "observed" data
        - 'bold_clean': clean_bold_downsampled (T_TR, N) — noise-free
        - 'bold_fine': clean_bold (T_fine, N) — fine-grained pre-downsampling
        - 'neural': x (T_fine, N) — neural activity
        - 'hemodynamic': dict with s, f, v, q each (T_fine, N) — in linear space
        - 'times_fine': t_eval (T_fine,)
        - 'times_TR': downsampled time points (T_TR,)
        - 'params': dict with A, C, hemo_params, SNR, TR, duration, solver
        - 'stimulus': the PiecewiseConstantInput or stimulus dict

    `def make_block_stimulus(n_blocks, block_duration, rest_duration, n_inputs=1, dtype=torch.float64) -> dict`:
    - Convenience function to create standard block-design stimuli.
    - Returns dict with 'times' and 'values' suitable for simulate_task_dcm.
    - Example: n_blocks=10, block_duration=30, rest_duration=20 creates alternating 30s ON / 20s OFF for total 500s.

    `def make_random_stable_A(n_regions, density=0.5, strength_range=(0.0, 0.3), self_inhibition=0.5, seed=None, dtype=torch.float64) -> torch.Tensor`:
    - Convenience function to generate random stable A matrices for simulation.
    - Diagonal: -self_inhibition (default -0.5 Hz, matching SPM default).
    - Off-diagonal: sparse (controlled by density), values in strength_range.
    - Returns the parameterized A (NOT A_free), ready for use in simulator.
