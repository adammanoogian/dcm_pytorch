```python
def profile_svi_step(
    model: Callable,
    guide: Callable,
    elbo: pyro.infer.ELBO,
    model_args: tuple,
    n_steps: int = 10,
) -> dict[str, Any]:
```

- Implements Option C from research Section 6.2: profile a few steps after training
- For each of n_steps:
  a. Time forward model: `poutine.trace(model).get_trace(*model_args)` with time.perf_counter
  b. Time guide: `poutine.trace(guide).get_trace(*model_args)` with time.perf_counter
  c. Time ELBO + backward: `loss = elbo.differentiable_loss(model, guide, *model_args); loss.backward()` with time.perf_counter
  d. Zero gradients after each step to avoid accumulation
- Return dict with:
  - "forward_times": list of per-step forward times
  - "guide_times": list of per-step guide times
  - "gradient_times": list of per-step ELBO+backward times
  - "forward_median": median forward time
  - "guide_median": median guide time
  - "gradient_median": median gradient time
  - "total_median": sum of three medians
  - "forward_pct": forward_median / total_median * 100
  - "guide_pct": guide_median / total_median * 100
  - "gradient_pct": gradient_median / total_median * 100

```python
def plot_pareto_frontier(
    results: dict,
    output_dir: str,
    n_regions: int = 3,
    formats: tuple[str, ...] = ("png",),
) -> None:
```

- Data source: For each result key matching the requested n_regions, read result["time_list"] and compute np.median(result["time_list"]) for the x-axis, and np.median(result["rmse_list"]) for the y-axis. IQR error bars from np.percentile(result["time_list"], [25, 75]) and np.percentile(result["rmse_list"], [25, 75]).
- X-axis: median wall time per dataset (log scale)
- Y-axis: median RMSE
- One scatter point per (guide_type, variant) at the given n_regions
- Color by guide type using GUIDE_COLORS, marker shape by variant (o for spectral, s for task, ^ for rDCM)
- Compute Pareto front: sort by x, then walk through points keeping those where no other point has both lower x AND lower y. Connect Pareto-optimal points with a dashed line.
- Error bars: IQR on both axes (horizontal for time, vertical for RMSE)
- Label each point with guide type abbreviation
- Save as "pareto_frontier_N{n_regions}"
